# Bem Geladin - Site Institucional

Este repositório contém o site institucional da **Bem Geladin**, desenvolvido com uma identidade visual completa e um layout totalmente responsivo.

## 🧊 Sobre o Projeto
O site foi criado com foco em apresentar a marca de forma clara e profissional, destacando:
- Os produtos oferecidos pela empresa  
- A identidade visual da marca  
- Um design moderno e adaptado a todos os dispositivos  
- Uma seção de contato para facilitar a comunicação com clientes e interessados  

## 🚀 Tecnologias Utilizadas
- HTML5  
- CSS3
